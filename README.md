# AERO 689 Presentation Planning

## Course Structure: 14 Weeks × 50-minute Lectures
**Instructor**: Dr. Raktim Bhattacharya
**Assessment**: Homework (30%), Paper Review (10%), Midterm (20%), Final Project (30%), Participation (10%)

### Presentation Framework
Each weekly presentation will include:
- **Opening Hook** (5 min): Real aerospace problem/challenge
- **Concept Introduction** (15 min): ML theory with aerospace context
- **Hands-on Example** (20 min): Live coding/demonstration with aerospace data
- **Industry Application** (8 min): Real-world aerospace case study
- **Wrap-up & Next Week Preview** (2 min)

## Detailed Weekly Schedule

### Phase 1: Foundations (Weeks 1-6)
**Week 1: Introduction & Motivation**
- Hook: Aircraft accidents due to poor data interpretation
- Content: Why ML for aerospace? Historical context, modern applications
- Demo: Basic data visualization with flight recorder data
- Case Study: Boeing 737 MAX MCAS system analysis

**Week 2: Linear Regression**
- Hook: Predicting aircraft fuel consumption for flight planning
- Content: Simple/multiple regression, least squares, gradient descent
- Demo: Drag coefficient prediction from wind tunnel data
- Case Study: Aircraft performance modeling for airlines

**Week 3: Model Evaluation & Regularization**
- Hook: Why do aerodynamic models fail in real flight conditions?
- Content: Cross-validation, bias-variance tradeoff, Ridge/Lasso
- Demo: Overfitting in airfoil pressure distribution modeling
- Case Study: NASA wind tunnel data validation techniques

**Week 4: Classification**
- Hook: Automatic flight regime detection for pilots
- Content: Logistic regression, decision trees, SVM
- Demo: Flight phase identification from sensor data
- Case Study: Aircraft fault detection systems

**Week 5: Clustering & PCA**
- Hook: Grouping similar aircraft configurations for design
- Content: K-means, hierarchical clustering, principal components
- Demo: Aircraft configuration clustering from design parameters
- Case Study: Airspace sector optimization using flight patterns

**Week 6: Dimensionality Reduction (POD Connection)**
- Hook: Reducing complex CFD data to essential features
- Content: PCA, SVD, proper orthogonal decomposition
- Demo: Modal analysis of wing vibration data
- Case Study: Flow field compression for real-time simulation

### Phase 2: Advanced Methods (Weeks 7-12)
**Week 7: Neural Network Basics**
- Hook: Learning complex aerodynamic relationships
- Content: Perceptrons, backpropagation, activation functions
- Demo: Nonlinear lift curve modeling
- Case Study: Engine performance prediction

**Week 8: Midterm Review & Exam**
- Review: Weeks 1-7 concepts with aerospace examples
- Exam: Closed book, covers foundational ML concepts

**Week 9: Deep Networks & CNNs**
- Hook: Analyzing satellite imagery for weather prediction
- Content: Deep architectures, convolution, pooling
- Demo: Aircraft detection in aerial imagery
- Case Study: Automated runway inspection systems

**Week 10: Sequence Models (RNN/LSTM)**
- Hook: Predicting aircraft trajectories for air traffic control
- Content: RNNs, LSTM, sequence-to-sequence models
- Demo: Flight path prediction from radar data
- Case Study: Predictive maintenance using sensor time series

**Week 11: Physics-Informed & Hybrid Models**
- Hook: Combining centuries of aerodynamics knowledge with ML
- Content: PINN, domain knowledge integration, hybrid approaches
- Demo: Physics-constrained airfoil optimization
- Case Study: NASA's physics-informed turbulence modeling

**Week 12: Robustness & Uncertainty**
- Hook: Safety-critical decisions in aerospace systems
- Content: Uncertainty quantification, adversarial robustness
- Demo: Confidence intervals in landing distance prediction
- Case Study: Autonomous aircraft decision-making under uncertainty

### Phase 3: Application (Weeks 13-14)
**Week 13: Project Development**
- Hook: Real-world aerospace ML challenges
- Content: Project methodology, data selection, validation strategies
- Demo: Student project presentations (work-in-progress)
- Case Study: Industry collaboration examples

**Week 14: Project Presentations & Wrap-Up**
- Hook: The future of ML in aerospace
- Content: Student final project presentations
- Demo: Live project demonstrations
- Case Study: Course reflection and next steps

### Technical Setup
- **Format**: Jupyter notebooks with embedded slides (using RISE extension)
- **Interactive Elements**: Live Python demonstrations
- **Visual Style**: Professional aerospace theme with equations, diagrams
- **Code Examples**: Progressively building complexity

### Aerospace Application Domains
1. **Aerodynamics**: Drag prediction, lift optimization, flow classification
2. **Flight Dynamics**: Trajectory prediction, stability analysis, control systems
3. **Propulsion**: Performance modeling, fault detection, optimization
4. **Structures**: Fatigue analysis, damage detection, material properties
5. **Operations**: Flight planning, scheduling, predictive maintenance
